# 踏切アプリ (Fumikiri App)

小学2年生の子供が喜ぶリアルな日本の踏切シミュレーターです。

## 特徴

- **リアルな踏切**: 日本の踏切を忠実に再現（X字標識、警告灯、遮断機）
- **4種類の電車**: 普通電車、特急、新幹線、蒸気機関車
- **リアルな音**: 踏切ベル、電車走行音、警笛をWeb Audio APIで合成
- **ゲーム要素**: 踏切が開いている間に人や車を安全に渡らせる
- **住宅街の背景**: リアルな日本の住宅街風景

## 遊び方

1. 電車の種類を選択
2. 「でんしゃをよぼう！」ボタンで電車を呼ぶ
3. 踏切が開いている間に左側のボタンで人や車を渡らせる
4. 電車通過中に踏切内にいると危険！

## 技術スタック

- Next.js 16
- React 19
- TypeScript
- Tailwind CSS
- Web Audio API

## 開発

```bash
npm install
npm run dev
```

http://localhost:3000 でアプリが起動します。

## デプロイ

このアプリはRenderで簡単にデプロイできます。

1. GitHubにリポジトリをプッシュ
2. Renderでリポジトリを接続
3. 自動的にビルド・デプロイされます
